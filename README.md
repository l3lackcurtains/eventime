# Eventime Frontend

Eventime is a time tracking app for freelancers, built with ReactJS, Ant Design, and GraphQL.

## Features

- Track time spent on projects
- Manage project billing
- Add and manage expenses
- User authentication and dashboard

## Technologies Used

- ReactJS
- Ant Design
- GraphQL
- Apollo Client
- Styled Components
